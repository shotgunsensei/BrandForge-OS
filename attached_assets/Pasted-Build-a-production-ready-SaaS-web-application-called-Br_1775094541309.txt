Build a production-ready SaaS web application called “BrandForge OS” (working title), a multi-tenant AI-powered marketing and advertising platform that helps businesses go from zero to hero with branding, design, campaigns, content creation, and performance optimization.

PRIMARY PRODUCT GOAL
The app should guide a user through the full marketing lifecycle:
1. Brand setup
2. Audience and offer definition
3. Campaign planning
4. Creative generation
5. Landing page / ad asset creation
6. Social/email/ad publishing preparation
7. Analytics and optimization recommendations

CORE POSITIONING
This is not just a design tool and not just an AI copywriter.
It is a business marketing operating system for small businesses, creators, consultants, agencies, and product owners.

TARGET USERS
- Solo founders
- Small businesses
- Agencies
- Multi-brand operators
- SaaS founders
- E-commerce brands
- Service businesses

TECH STACK
Choose a modern, scalable stack that works well in Replit and can be productionized:
- Frontend: React + TypeScript + Tailwind + shadcn/ui
- Backend: Node.js + TypeScript
- Database: PostgreSQL with Prisma ORM
- Auth: secure tenant-aware authentication
- Storage: object storage abstraction for tenant files/assets
- Payments: Stripe subscriptions + plan management
- AI: pluggable AI service layer for copy, strategy, and recommendations
- Charts/analytics: clean dashboard visualizations
- Queue/background architecture where appropriate
- Build with modularity and future API access in mind

MULTI-TENANT REQUIREMENTS
- True tenant isolation at the database layer
- Every account belongs to a tenant/workspace
- Support multiple users per tenant
- Role-based access control:
  - Owner
  - Admin
  - Editor
  - Viewer
  - Client
- Tenant-scoped assets, brand kits, campaigns, designs, analytics, and billing
- Prepare architecture for future white-label and custom domains
- Include audit logs for important actions
